To run the example from the book, you'll need to set up your environment.
Get Openfire 3.6.x from http://www.igniterealtime.org/downloads/
Read the installation instructions and install Openfire
During the installation you will be asked to enter domain name for the Openfire server. Make sure the domain name is "localhost" or the tests will fail.
After the installation restart the server and login to the admin console using the username "admin" and the password that you entered during the installation.
